## How To Use

<!-- Example: -->

To clone and run this application, you'll need [Git](https://github.com/cristhian420/country-quiz-devChallenge) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/cristhian420/country-quiz-devChallenge

# Install dependencies
$ npm install

# Run the app
$ npm start
```
